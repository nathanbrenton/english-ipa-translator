# English IPA Translator

A small web application that converts English words and phrases into IPA-like pronunciation output.

This project is the first milestone of a larger lyric-analysis and lyric-generation portfolio project. The first version focuses on deterministic pronunciation lookup and ARPABET-to-IPA conversion.

## Planned MVP

- React front end
- FastAPI back end
- CMUdict-style pronunciation lookup
- ARPABET-to-IPA conversion
- Structured word-by-word translation results
- Graceful unknown-word handling
- Back-end tests for pronunciation logic

## Current Status

Project initialized. Back-end and front-end implementation coming next.
